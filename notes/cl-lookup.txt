Соответствие между версиями Visual Studio и наборами инструментов (Toolset) строится по определенной логике, 
где начиная с 2015 года мажорная версия инструментов зафиксирована на уровне 14, а меняется только минорный номер.

В старых выпусках номер набора инструментов (Platform Toolset) совпадал с внутренней версией Visual C++:
v90 — Visual Studio 2008.
v100 — Visual Studio 2010.
v110 — Visual Studio 2012.
v120 — Visual Studio 2013.

VCPKG_PLATFORM_TOOLSET - ключевая переменная для указания версии MSVC:
v141 - VS 2017
v142 - VS 2019
v143 - VS 2022
v144 - VS 2025 (предварительно)
